# VOLT Remote App

A premium Flutter TV remote control application matching the Kinetic Luxury HTML/CSS design mockups.

## Features Implemented
- **Splash Screen** & **Setup Wizard** (Steps 1-3)
- **Main Shell** with custom glassmorphic bottom navigation
- **Remote Control Screen** featuring a complex multidirectional touchpad and tactile controls
- **Apps Library** with spotlight bento grids and squircle app icons
- **Cast Dashboard** with active device tracking and media library
- **Settings Screen** with profile, device info, and system configuration
- **Voice Search** interface with concentric visualizers

## How to Run

Since the `flutter` command was not found in your system's PATH during generation, the project was manually scaffolded. 

1. Ensure [Flutter SDK](https://docs.flutter.dev/get-started/install) is installed and added to your system PATH.
2. Open this directory (`volt_remote_app`) in your terminal or IDE (VS Code/Android Studio).
3. Run `flutter pub get` to download dependencies.
4. Run `flutter run` to launch the app on your preferred emulator or device (Chrome is perfect for quick visual testing).
